* ABOUT
This is a repository for my personal Emacs config, which is based on Derek Taylor's
custom configuration, with some additions and tweaking such as compatibility with
Windows and support for the upcoming lexical-binding default mode in Elisp.

* INSTALLATION
1. Remove "~/.emacs.d", "~/.emacs", and any other config files and directories
auto-generated by Emacs on your home folder.
2. Create the directory "~/.config/emacs" on your home folder.
3. Clone this repository in "~/.config/emacs".
4. If you're on a Windows system, you need to create an environment variable
for Emacs to look for your init.el file in the "~/.config/emacs" directory.
See the next section on how to set this on Windows and then proceed with step 5.
5. Run Emacs.

** Setting XDG Environment Variable On Windows
1. Once you removed "~/.emacs.d" and "~/.emacs" from your home directory,
go to "C:\Users\{YourUsername}\AppData\Roaming" and delete any ".emacs.d" or ".emacs"
file or directory from there as well.
2. Press Win + R, type sysdm.cpl and hit Enter
3. Go to the Advanced tab and click Environment Variables... at the bottom.
4. Under User Variables, click New... and add the following variables:

   | Variable name   | Variable value            |
   | HOME            | "C:\Users\{YourUsername}" | # Omit this if it already exists
   | XDG_CONFIG_HOME | "%HOME%\.config"         |

5. Run Emacs.
6. Verify Emacs is reading from the correct path:
   6.1. Press M-x
   6.2 Type describe-variable and hit Enter
   6.3 Type user-init-file and hit Enter
   6.4 You should see the exact following path:
       "C:\Users\{YourUsername}\.config\emacs\ini.el"
